HTML Favicon is a small icon that represents a website and appears in the browser’s tab or bookmark bar. It is defined in HTML using the <link> 
tag with the rel attribute set to “icon.” The Favicon is also known as the Favorite icon.

Example of favicon tag<!DOCTYPE html> 
<html> 
  
<head> 
    <meta charset="utf-8" /> 
  
    <title>HTML Favicon</title> 
  
    <!-- Add icon link -->
    <link rel="icon" href= 
"https://media.geeksforgeeks.org/wp-content/cdn-uploads/gfg_200X200.png" 
        type="image/x-icon"> 
</head> 
  
<body> 
    <h1 style="color:green;"> 
        GeeksforGeeks 
    </h1> 
  
    <p> 
        Welcome to my website 
    </p> 
</body> 
  
</html>
